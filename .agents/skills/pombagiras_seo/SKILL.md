# POMBAGIRAS ECOSYSTEM - SEO & Metadata Skill

Esta skill especializada aplica-se a tarefas de otimização de mecanismos de busca (SEO), auditoria de metadados, links de canonical e estruturação de dados de autoridade.

## 1. Schema.org e JSON-LD
- Toda página principal deve conter seu bloco de dados estruturados em JSON-LD (`application/ld+json`).
- O Schema deve representar de forma precisa as entidades de `Person` (como Alexia Melusine), `ProfilePage`, `Quiz`, ou `WebSite`.
- Sempre preserve e valide o link da Wikidata do autor (`https://www.wikidata.org/wiki/Q139714039`).

## 2. Meta Tags de Compartilhamento
- Garanta que as tags Open Graph (`og:*`) e Twitter Card (`twitter:*`) estejam corretas e sincronizadas.
- O link do banner deve seguir estritamente o mapeamento de imagens do ecossistema.

## 3. Crawlers, Canonical e hreflang
- Respeite e preserve a sinalização multilíngue (`hreflang` / `x-default`) e a tag canonical em todas as páginas para evitar punições por conteúdo duplicado.
- Não altere as diretivas de rastreamento do `robots.txt` sem alinhamento expresso.
